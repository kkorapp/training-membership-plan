# Training Membership Plan System

This project is a Java console application for managing different types of training membership plans.

## Implemented Concepts
- Abstract class (MembershipPlan)
- Interfaces (Billable, RemoteAccess, Freezable)
- Inheritance
- Polymorphism
- Method overriding

## Classes
- MembershipPlan (abstract)
- GymMembershipPlan
- PersonalTrainingPlan
- OnlineCoachingPlan
- CorporateWellnessPlan

## Features
- Calculates monthly net and gross prices
- Calculates total contract value
- Supports polymorphic operations
- Uses instanceof for additional capabilities

## How to run
Run Main.java to see example output.
